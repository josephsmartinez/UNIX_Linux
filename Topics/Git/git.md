# Git and Git Management


> git init
> git add [filename]
> git commit -m "Adding files"
> git status

## Configure Username and User Email

> git config --global user.name johndoe
> git config --global user.email

## Reset git head

Soft reset
> git reset HEAD
Undo `git --reset`
> git reset 'HEAD@{1}'
