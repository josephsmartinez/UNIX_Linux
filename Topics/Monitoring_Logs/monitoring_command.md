> top
System running time
> uptime
Processes
> ps
Process tree
> pstree
Processor Usage
> mpstat
CPU utilization and I/O statistics
> iostat
Display and collect information about system activity
> sar
Information Non-Uniform Memory Architecture 
> numastat
Information about all system calls a process makes
> strace
