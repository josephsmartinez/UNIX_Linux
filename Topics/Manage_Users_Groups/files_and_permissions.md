## Permissions with letters

chmod with Letters

Usage: chmod {options} filename

| Options | Definition          |
| u       | owner               |
| g       | group               |
| o       | other               |
| a       | all (same as ugo)   |
| x       | execute             |
| w       | write               |
| r       | read                |
| +       | add permission      |
| -       | remove permission   |
| =       | set permission      |
