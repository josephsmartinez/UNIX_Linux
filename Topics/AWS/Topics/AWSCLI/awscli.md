# Amazon Command Line Interface

> apt install awscli

## Configure AWS CLI

> export AWS_ACCESS_KEY_ID=$AWSAccessKeyId
> export AWS_SECRET_ACCESS_KEY=$AWSSecretKey
> export AWS_DEFAULT_REGION=us-west-2
> aws configure
>
